# Bold Dragoon CC Hub — Professional Edition 3.0

This release builds on the working Netlify + Supabase setup already deployed.

## New in 3.0
- Professional mobile dashboard and quick actions
- Personal player profiles with club roles
- Team-filtered fixture list and ground directions
- Improved one-tap availability and live squad totals
- Published team sheet with copy-to-clipboard
- Audience and priority settings for club notices
- Clearer player directory and polished mobile UI

## Deploy
Upload all files and folders in this package to the root of the existing GitHub repository and commit directly to `main`. Netlify will redeploy automatically. Keep the existing Netlify environment variables and Supabase project.
